## рҹҡҖ Current Mission: Wallt

**[Wallt](https://wallt.in)** is India's most comprehensive **legacy planning and digital vault platform**.

As Co-founder & CTO, I'm addressing a critical gap in the financial ecosystem: the fragmentation of digital wealth and the complexity of inheritance.

### рҹҺҜ The Problem

- рҹ“ұ **Digital Asset Fragmentation** вҖ” Assets scattered across 50+ platforms with no unified view
- рҹ”җ **Inaccessible Credentials** вҖ” Families lose access to millions because credentials die with the owner
- вҸ° **Delayed Wealth Transfer** вҖ” Inheritance in India is archaic, often taking months of legal battles

### рҹ’Ў Our Solution

| Feature | Description |
|---------|-------------|
| **рҹҸҰ Digital Vault** | Bank-grade encrypted repository for documents, credentials, and financial assets |
| **рҹӨқ Smart Handover** | A proprietary "dead man's switch" ensuring information reaches nominees *only* when exigency criteria are met |
| **рҹ”’ Zero-Knowledge Security** | Architected with AWS Nitro Enclaves + KMS вҖ” even we cannot access user data |
| **рҹ‘Ҙ Smart Nominees** | Granular access control вҖ” designate specific assets to specific people |

- **Semantic reranking** вҖ” Claude reranks vector search results for precision file selection
